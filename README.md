# Mr. Whiskers' Blog Page

A simple HTML project built as part of my coding practice.

## About the Project

This project is a basic blog-style webpage for **Mr. Whiskers' Blog Page**. It includes an introduction, navigation links, blog post sections, and contact details, all structured using semantic HTML.

The page is designed to practise building a multi-section webpage with clear organization and meaningful HTML elements.

## Features

- Header with page title and featured image
- Navigation menu with internal page links
- About section
- Blog posts section with multiple articles
- Contact section with phone and email links
- Use of semantic HTML elements such as `header`, `main`, `section`, `article`, `footer`, `figure`, and `address`

## Built With

- HTML5

## What I Learned

While building this project, I practised:

- Creating a basic HTML document
- Structuring a webpage with semantic HTML
- Using navigation links with section IDs
- Adding images with captions
- Organising content with sections and articles
- Using the `address` element for contact information
- Creating clickable phone and email links
- Building a simple blog-style layout
